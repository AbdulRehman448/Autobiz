# AutoBiz-automation
AutoBiz is a business workflow automation project built using n8n.  
The goal of this project is to automate repetitive business tasks such as collecting customer leads, sending responses, and notifying teams automatically.

## Project Goal
Many small businesses still handle customer inquiries manually.  
AutoBiz demonstrates how automation can replace manual tasks like:

- collecting customer data
- sending confirmation emails
- notifying internal teams
- managing leads
- 
The system uses workflow automation to handle these processes automatically.
